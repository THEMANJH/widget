1. 회원가입 후 API 키 받기

2. 코드 복사하기

대시보드에서 아래와 같은 코드를 복사하세요:

<script> (function() { const script = document.createElement('script'); script.src = 'https://widget-seven-taupe.vercel.app/public/loader.js'; script.setAttribute('data-api-key', 'YOUR_API_KEY_HERE'); document.head.appendChild(script); })(); </script>

3. 블로그에 붙여넣기

티스토리: 관리 → 꾸미기 → 스킨 편집 → HTML 편집에서 </body> 태그 위에 붙여넣기

네이버 블로그: 블로그 관리 → 디자인 설정 → 레이아웃/위젯 설정에서 HTML 위젯 추가
